# Ingestion Checklist

Use this checklist before running `tools/ingest.py`.

1. **Isolate the System (Definition)**
   - Define the exact system boundary and entropy quantity in scope.
2. **Identify Assumptions**
   - Record all modeling assumptions and epistemic status.
3. **Define Must-Fail Falsification Tests**
   - Specify concrete failure conditions that must reject the claim/model.
4. **Map Citations to `refs.yaml`**
   - Ensure every citation ID used in the draft exists in `atlas/bibliography/refs.yaml`.
5. **Formulate Claims**
   - State the claim clearly, bind evidence/provenance, and confirm domain/relation references.
